## Project description

ApparenceKit Supabase Boilerplate — a Flutter starter template generated by the ApparenceKit CLI. Provides authentication, subscription management (RevenueCat), notifications, onboarding, feedback/feature voting, and settings modules.

**Current state:** The code uses Firebase (firebase_auth, cloud_firestore, firebase_messaging) for backend services. Supabase SQL migrations are provided in `supabase/migrations/`. The intent is to migrate backend calls to Supabase.

**Architecture:** 3-layer (API → Repository → Provider/UI) with Riverpod for state management, Freezed for immutable models, and go_router for navigation.

**Key commands:**
- `flutter pub get` — install dependencies
- `dart run build_runner build --delete-conflicting-outputs` — generate .g.dart and .freezed.dart files
- `flutter analyze lib test` — run static analysis
- `flutter test` — run tests
